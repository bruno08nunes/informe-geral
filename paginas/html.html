O que é HTML?

HTML é uma linguagem de marcação em hipertexto. Ela é a estrutura de uma página web, definindo seu conteúdo. Por exemplo, no código podem ir parágrafos, títulos, tabelas ou listas. Ela também é responsável pelo significado da página, com ela tendo que estar de um jeito coerente.

Diferente do que muita gente pensa, HTML não é considerado como uma linguagem de programação. Isso se deve ao fato de ela não ser funcional, não podendo criar funcionalidades dinâmicas. Essa linguagem não segue as completudes de Turing, que define o que uma linguagem de programação deve conseguir fazer. Essas capacidades são:

  • Mudar informações armazenadas em uma memória;
  • Fazer cálculos;
  • Mudar a ordem de execução;
  • Tomar decisões.

Você pode estar pensando que essa linguagem é inútil então, porém é muito pelo contrário. Ela faz a estrutura de um site, mostrando os conteúdos na tela. Inclusive, esse site inteiro foi programado utilizando HTML (também foi usado CSS e JS, que você pode ver mais sobre nas respectivas páginas).

Um código HTML usa elementos, com tags de abertura, tags de fechamento, conteúdo e parâmetros. Apenas a tag de abertura é obrigatória em todos os elementos. A sintaxe é formada pelas tags dentro dos sinais < >. A anatomia de um código HTML segue o padrão:

<!DOCTYPE html>
<html>
<head>
</head>
<body>
</body>
</html>

Explicando um pouco o código, ele é formado por uma tag sem fechamento (chamada de tag vazia) que indica que o que virá a seguir é um código HTML. Depois vem a tag <html> que conterá todo o conteúdo da página dentro dele. No elemento <head> deve ir todas as informações não visíveis para os usuários e no <body> deve ir os elementos que os usuários poderão ver.

Aquela é só a base do código, com muitas informações podendo ser adicionadas a eles. No código abaixo criaremos um título (com a tag <h1>), um parágrafo (com a tag <p>) e colocaremos uma imagem (com a tag <img>).

<!DOCTYPE html>
<html>
<head>
</head>
<body>
    <h1>Título</h1>
    <p>Parágrafo</p>
    <img src="imagem.jpg" alt="Imagem de Teste">
</body>
</html>

Essa página não tem como objetivo lhe ensinar a como estruturar um HTML, apenas a como ele funciona. Caso deseje aprender mais sobre, há diversos cursos sobre como os da MDN, da W3School ou do Curso em Vídeo.
