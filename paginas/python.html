O que é Python?

Python é uma linguagem de programação. Ele pode ser utilizado para desenvolvimento de software, modificar arquivos e até mesmo para o desenvolvimento web. Caso tenha visto as páginas explicando sobre JavaScript, não há uma forma de ligar o python ao HTML de forma tão simples (sem usar uma biblioteca de código)

Python é considera uma das linguagens de programação com a sintaxe mais simples. Ela utiliza como padrão a lingua inglesa. A linguagem não vem como padrão instalada no computador, sendo necessário instalá-lo. Atualmente está na versão 3.

Mostraremos abaixo alguns exemplos do que a linguagem é capaz de fazer:

Calcular valores

num1 = int(input())
num2 = int(input())
print(num1 + num2)

Esse código guarda dois valores em diferentes variáveis e depois os soma.

Comparar se um valor é maior que outro

num1 = int(input())
num2 = int(input())
if num1 > num2:
    print("O primeiro é maior")
elif num2 > num1:
    print("O segundo é maior")
else:
    print("Os dois valores são iguais")

Esse código pega dois valores, comparando se um é maior que o outro e mostrando se é ou não na tela.

Gerar um gráfico

import matplotlib.pyplot as plt

x = [11, 5, 5, 3] #Dados do gráfico
myLabels = ["Maçã", "Uva", "Laranja", "Pera"] #Nome de cada parte do gráfico
plt.pie(x, labels = myLabels) #Cria o gráfico
plt.show()

Esse código utiliza a biblioteca Matplotlib, que gera gráficos com os dados inseridos. Caso queira aprender mais sobre a linguagem, há diversos cursos, como os da W3Schools ou do Curso em Vídeo.
